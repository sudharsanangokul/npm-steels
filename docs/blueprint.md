# **App Name**: NTM Metals - Steel Solutions

## Core Features:

- Product Catalog: Display HR, CR, GI, SS, and Japan steel sheets with detailed specs, images, and enquiry options.
- Online Enquiry System: Enable customers to request quotes for steel sheets and cutting services.
- Services Showcase: Present laser cutting, plasma cutting, CNC bending, welding, and custom fabrication with visuals and descriptions.
- Image Gallery: Display product images, cutting machines, finished components, on-site deliveries, and fabrication work.
- Contact & Support: Provide address, phone, WhatsApp, email, Google Maps, and support hours.
- Enquiry Cart: Allow users to add multiple products to an enquiry cart before submitting.

## Style Guidelines:

- Primary color: Steel Blue (#1E2A36) to establish a professional and industrial feel.
- description: Background color: Metallic Dark (#2A2F33), a desaturated variant of the primary, to keep the industrial vibe.
- Accent color: Accent Blue (#6B8EA3), which provides a visually interesting and appropriate highlight.
- Font pairing: 'Montserrat' sans-serif for headings and 'Poppins' geometric sans-serif for body text to ensure clean headings and high readability. Note: currently only Google Fonts are supported.
- Use minimal, geometric icons to represent services and product features, aligning with the industrial aesthetic.
- Employ a spacious layout with 60–90px section spacing for a clean, modern feel; implement a responsive grid system with 3–4 columns for product listings.
- Implement smooth hover effects on product cards and buttons; use subtle fade-in and slide-up animations for section entrances.
