# **App Name**: NTM Metals - Steel Supply & Cutting

## Core Features:

- Product Catalog: Display a catalog of HR, CR, GI, SS, and other steel sheets with details and images.
- Quote Request: Allow users to request quotes for specific products, dimensions, and quantities.
- Service Showcase: Showcase laser cutting, plasma cutting, CNC bending, welding, and custom fabrication services.
- Image Gallery: Display images of products, machinery, and fabricated works in a gallery.
- Contact Form: Enable users to send inquiries via a contact form, including company details and file uploads.
- Enquiry Cart: Allow users to add multiple products to an enquiry cart before submitting a quote request.
- AI Quote Tool Assistant: Integrate an AI tool to assist customers in determining the precise product dimensions for optimal manufacturing using an advanced product selector.

## Style Guidelines:

- Primary color: Steel Blue (#1E2A36) to convey industrial strength and trustworthiness.
- Background color: Very light desaturation of Steel Blue (#E6E7E9).
- Accent color: Accent Blue (#6B8EA3) for highlights and interactive elements to draw attention without overwhelming the palette.
- Body font: 'Montserrat', sans-serif, for a modern and readable feel across the website.
- Headline font: 'Poppins', sans-serif, for clean and bold headings.
- Use crisp, clean icons related to steel products and services, with a metallic touch.
- Modern and clean layout with ample spacing between sections (60-90px).
- Card-based product grid with rounded corners (10-20px radius) and soft shadows.
- Smooth scroll, animated section entrances (fade-in, slide-up), and subtle hover effects to enhance UI.
